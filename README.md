# CPProguelike
Roguelike game engine made with SDL2

## Dependencies
1. SDL >= 2.0
2. SDL2_image
3. SDL2_ttf >= 2.0
4. g++ >= 5.4
5. cmake >= 3.7+
6. make >= 4.1

## Build Instructions
1. From terminal, run `mkdir build && cd build`
2. From terminal, run `cmake .. && make`
3. From terminal, run `./WallsEngine`
